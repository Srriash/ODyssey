Run analysis with example_run1.xlsx

01

Upload the file

Choose example_run1.xlsx from your local machine.

  • Use the main uploader in the Analysis section (not the Compare Runs uploader).
  • If you previously loaded a config, clear it to start fresh for this walkthrough.
02

Select the sheet + time column

Pick Sheet1, confirm the time column, and set the time unit.

  • Time column should be the left-most column in the file.
  • Use minutes or hours depending on the scale you want reported.
  • If the time is formatted as hh:mm:ss, choose the hh:mm:ss option and it will be converted.
03

Blank normalization (optional)

If the data is not blank-normalized, select one or more blank columns.

  • Pick the columns that contain blank wells so OD is baseline-corrected.
  • Leave blank normalization off if the input is already corrected.
04

Map treatments + replicates

Group columns into treatments, then add each replicate group.

  • Select one or more columns that belong to the same treatment.
  • Set the replicate number for those columns, then click Add replicate group.
  • Repeat until every data column is assigned to a treatment + replicate.
05

Make the plots

Choose overlay or small multiples, toggle SD bands, and set labels.

  • Use global labels to set the plot title and axis labels.
  • Optional: create plot groups to compare subsets of treatments.
  • Enable custom tick intervals if you want fixed spacing on axes.
06

Fit and export

Confirm the fit window, run analysis, and download the zip export.

  • Adjust the highlighted fit window to cover the exponential phase.
  • Optionally calculate live R2 to evaluate window quality.
  • Run analysis, review the results table, then build the download zip.
  • For comparisons later, include Long format CSV in the download.
  • Keep the generated zip file; you will upload it in the Compare Runs section.

Compare runs using example_run1.xlsx + example_run2.xlsx

01

Analyze both runs

Run analysis on each file and export the zip from Downloads.

  • Repeat the same settings for both files to make results comparable.
  • Make sure Long format CSV is enabled in both exports.
02

Upload zip exports

Use the Compare Runs section to load the zip files.

  • Upload both zip files together to compare in one view.
  • If prompted, pick a target time unit for the comparison.
03

Review metrics

Inspect grouped bar plots to compare growth rate, doubling time, and AUC.

  • Optionally rename treatments so labels match across runs.
  • Use the summary table to compare run-level averages.
04

Overlay growth curves

Select treatments and show SD bands to compare curves directly.

  • Select the same treatments for each run to compare shapes.
  • Use SD bands to see replicate variability.
