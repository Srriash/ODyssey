Example Run

Practice the full workflow with a ready-made dataset.

Download the example files, run a full analysis, and then compare the two runs using the comparison workflow.

Run analysis with example_run1.xlsx

01

Upload the file

Upload example_run1.xlsx using the main Analysis uploader.

02

Select the sheet + time column

Pick Sheet1, set the time column to the first column, and choose minutes.

03

Blank normalization

Keep OD values are blank normalized checked. No blank columns are needed for this dataset.

04

Map treatments + replicates

Create replicate groups by treatment.

  • Select treatmentA_rep1, treatmentA_rep2, and treatmentA_rep3, then click Add replicate group.
  • The app will auto-suggest a treatment name; edit it if you want.
  • Repeat the same process for the other two treatments.
05

Fit window

Set the exponential window manually to 170–240 minutes.

  • Drag the window handles until the highlighted range reads 170 to 240.
  • Use the same window for both example runs.
06

AUC settings

Do not change the AUC settings for this example.

  • Leave AUC unit on the automatic default.
  • Leave AUC range on the default option.
  • Leave growth rate and doubling time units on their defaults.
07

Run analysis

Run the analysis after setting the window.

08

Make the plots

Set plot mode to Overlay (compare treatments).

  • Rename the global plot title to Example run 1.
  • Leave SD bands on (each treatment has three replicates).
09

Export

Download the zip export.

  • Keep Results CSV and Config JSON enabled.
  • Enable Long format CSV.
  • Build the zip and save it as example_run1.zip.
10

Prepare for comparison

Repeat the same settings on example_run2.xlsx to make the runs comparable.

  • Keep Results CSV enabled.
  • Enable Long format CSV.

Compare runs using example_run1.xlsx + example_run2.xlsx

01

Analyze both runs

Repeat the same steps for example_run2.xlsx.

  • Use Sheet1, time in minutes, and blank-normalized data.
  • Set the exponential window to 170–240 minutes.
  • Export with Long format CSV enabled.
  • Save the zip as example_run2.zip.
02

Upload zip exports

Upload example_run1.zip and example_run2.zip in Compare Runs.

  • Select minutes as the target time unit.
03

Review metrics

Use the comparison table and summary to compare the two runs.

  • Select the metrics you want to compare.
04

Overlay growth curves

Plot the same treatments for both runs to compare curve shapes.

  • Select both runs.
  • Select all treatments.
  • Leave SD band off (single-replicate example data).
05

Download comparison data

If you want to download comparison or summary data, use the individual download buttons.
